Project Title
Rock-Paper-Scissors Game in C
Functional Requirements
The program must allow the user to play Rock-Paper-Scissors against the computer.

The computer should randomly select between Rock, Paper, and Scissors.

The program must:

Accept user input (1 for Rock, 2 for Paper, 3 for Scissors).

Compare user choice with computer choice.

Display the result (Win, Lose, or Draw).

The game should continue until the user decides to quit.
Features
Simple and interactive command-line interface.

Randomized computer moves using rand() function.

Clear instructions for user input.

Displays both user and computer choices.

Shows the outcome of each round.

Option to play multiple rounds until the user exits.
How to run the project:
Play the game:

Enter 1 for Rock, 2 for Paper, 3 for Scissors.

The computer will generate its choice.

The result will be displayed.

Continue playing until you choose to quit.
